Moo
===

_An album-forward, read-only music player_


[![moo](screenshot.png)](screenshot.png)


Choose your music folder:

```
Moo/play/config.py:

BASE ='/Users/steve/Music'
```

Clone the app:

```
$ git clone https://github.com/siznax/moo.git
```

Start the app:

```
$ cd moo
$ ./play.sh
# visit localhost:5000 in your browser
```
