moo Components

Visual components roughly map to Jinja2 templates in moo/Moo/play/templates/*.html.

album
displays selected album cover, audio controls, tracklist and metadata
alpha-buttons
displays A-Z links to artists from all albums
artists
displays artists/count from selected albums
clock
displays system clock time every second
covers
displays selected album covers
footer
displays application name and version
genre-buttons
displays genre buttons from moo/Moo/play/config.py
genres
displays genres/count from albums selected
header
displays facet, alpha, and genre buttons
index
displays header and selected albums
metadata
displays table of normalized media tags for current track
tracklist
displays table of normalized media tags for current track
