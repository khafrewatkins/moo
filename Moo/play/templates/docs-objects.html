moo Objects

A source code reference

album
map of album attributes (dict)
aldex
minimal map of album data (dict)
alkey
file path of an album (string)
app
python Flask application with default web server
artists
map of album artist names and count (dict)
base
file path of music library (string)
config
application constants (file)
genres
map of album genre labels and their counts (dict)
hearts
persistent list of "hearted" albums and tracks (flat file)
history
persistent map of tracks and played count (JSON file)
index
list of alkeys (list of strings)
labels
labels defined for genre-buttons (config file)
links
auto-generated links to external services like Discogs, Musicbrainz, etc. (text file)
mtags
normalized audio file media tags (APE, ID3, etc.) (dict)
names
names defined for artist-buttons (config file)
results
index of albums selected by search results (list of strings)
settings
persistent option values (browser cookie)
stars
persistent list of "starred" albums and tracks (file)
stations
user-defined map of radio station names and URLs for station-buttons (config file)
track
data describing a single audio file (dict)
videos
persistent map of album or artist video URLs (JSON file)
