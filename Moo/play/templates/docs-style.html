Example custom color classes 

/*
  Native values from Safari audio element (dark to light)
 
    1: rgb(36,36,36)    <= background
    2: rgb(76,76,76)    <= slider-right
    3: rgb(68,68,68)    <= slider-left
    4: rgb(78,78,78)    <= time remaining
    5: rgb(114,114,114) <= play button
    6: rgb(128,128,128) <= track position
 
  Highlight colors (shades of pink, "dark" to light)
 
    a: rgb(255,96,178)
    b: rgb(255,75,161)
    c: rgb(254,168,252)
*/

.bg-1 { background-color: rgb(36,36,36);    }
.bg-2 { background-color: rgb(76,76,76);    }
.bg-3 { background-color: rgb(135,135,135); }
.bg-4 { background-color: rgb(156,156,156); }
.bg-5 { background-color: rgb(227,227,227); }

.bg-a { background-color: rgb(255,96,178); }
.bg-b { background-color: rgb(255,75,161); }
.bg-c { background-color: rgb(254,168,252); }

.clr-1 { color: rgb(36,36,36);    }
.clr-2 { color: rgb(76,76,76);    }
.clr-3 { color: rgb(135,135,135); }
.clr-4 { color: rgb(156,156,156); }
.clr-5 { color: rgb(227,227,227); }

.clr-a { color: rgb(255,96,178); }

moo Symbols

See moo/Moo/play/lib.py

⏭ U+23ED Next Track Button
link to next track (emoji.next)
⏮ U+23EE Last Track Button
link to previous track (emoji.prev)
☆ U+2606 White Star
album or track star button (emoji.star)
♡ U+2661 White Heart Suit
album or track heart button (emoji.heart)
❔ U+2754 White question mark
link to documentation (emoji.help)
⭐ U+2B50 Star
album or track has been starred (emoji.starred)
🎲 U+1F3B2 Game die
link to random album (emoji.random)
🎵 U+1F3B5 Musical note
link to base index (emoji.base)
🎷 U+1F3B7 Saxophone
app icon, link to album covers (emoji.app)
💜 U+1F49C Purple Heart
album or track has been hearted (emoji.hearted)
😭 U+1F62D Loudly Crying Face
404 Page not found (emoji.notfound)
🚫 U+1F6AB Prohibited
link to albums missing metadata (emoji.none)
